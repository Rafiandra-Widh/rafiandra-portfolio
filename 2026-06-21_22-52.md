## The rebuilt app (`site/`)
- **Stack:** Next.js 16.2.9 (App Router, Turbopack) + TypeScript. **No Tailwind** — styling is inline styles + CSS-variable theming (`html[data-theme="light|dark"]`) in `app/globals.css`. Fonts via Google Fonts `<link>` (Hanken Grotesk, Newsreader italic, Poppins).
- **Routes:** `/` (home, uses source's *variant 3* hero), `/work`, `/about`, `/contact`, `/work/[slug]` (4 case studies: idds, ocx, legion, pijar — statically generated via `generateStaticParams`).
- **Key files:**
  - `lib/data.ts` — ALL copy + project metadata + case-study content (single source of truth). `projects[]` has `{num, slug, title, tag, year, role, grad, cover, stack}`.
  - `components/Effects.tsx` — custom cursor (desktop-only), magnetic buttons, scroll-reveal (IntersectionObserver), re-runs on route change.
  - `components/ThemeProvider.tsx` — light/dark, defaults to system preference (in-memory only).
  - `components/Nav.tsx`, `components/Footer.tsx`, `components/case-parts.tsx` (shared case sections), `components/cases/{Idds,Ocx,Legion,Pijar}.tsx`.
